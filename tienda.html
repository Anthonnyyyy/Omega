Flipper Zero

Descubre el poder de la computación
compacta con el Raspberry Pi, la mini
computadora ideal para estudiantes,
desarrolladores, entusiastas de la tecnología
y proyectos DIY.

Raspberry Pi

Controla señales infrarrojas, subGHz, RFID
/NFC y más. Conéctalo vía USB o Bluetooth y
llévalo a donde quieras. Ideal para makers,
pentesters y entusiastas del hardware.

Licencia Windows 11 pro

Windows 11 Pro es la versión profesional del sistema operativo de Microsoft
diseñada para quienes exigen alto rendimiento, seguridad reforzada y
herramientas avanzadas de gestión

Wifi Pineapple

El WiFi Pineapple es un dispositivo de auditoría de redes
inalámbricas diseñado para pruebas de
penetración (pentesting) en ciberseguridad.

USB Rubber Ducky

El Rubber Ducky, uno de los primeros dispositivos desarrollados con este enfoque, es capaz de emular un dispositivo de interfaz humana (HID) como un teclado para que el sistema sobre el que actuará considere que es confiable y acepte sus interacciones.
